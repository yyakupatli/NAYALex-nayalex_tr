# NAYALex --> nayalex_tr
Türkçe dil desteği sunan NAYALex Duygu Sözlüğü

38 farklı duygu için sonuç veren duygu sözlüğüdür.

# Hakkında
Bu sözlük(nayalex_tr), paylaşılan metinlerden daha fazla duygu yakalamak için metinden olumlu-olumsuz ile birlikte (umut, kaygı, sevgi, karamsarlık, iyimserlik, öfke, korku, üzüntü vb.) 38 farklı duygu çıkarımı yapan bir duygu sözlüğüdür. NRC Duygu Sözlüğüne(http://saifmohammad.com/WebPages/NRC-Emotion-Lexicon.htm) ve Plutchik’in Temel Duyguların Psikoevrimsel Teorisine dayandırdığımız sözlüğümüzün her bir kelimesi 38 farklı duygudan en az biri ile ilişkilendirilmiş  kelimelerden oluşmaktadır.
